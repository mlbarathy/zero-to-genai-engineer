# GenAI Pre-work: Python, Math & Neural Networks

> Complete this **before** attending the GenAI weekend session.
> Total time: ~3 hours | Self-paced | No prior experience needed

---

## Why This Exists

The GenAI course (M00 onwards) assumes you can read Python code and have a basic mental model of how neural networks learn.

This pre-work closes that gap — fast. It covers **only** what you need. Nothing extra.

---

## The 3 Notebooks

| # | Notebook | Time | What You Learn |
|---|---|---|---|
| 1 | `01_python_for_genai.ipynb` | ~60 min | Variables, lists, loops, dicts, f-strings |
| 2 | `02_math_intuition.ipynb` | ~60 min | Vectors, dot product, probability, softmax |
| 3 | `03_neural_networks_intuition.ipynb` | ~60 min | How a neural network learns — in plain English |

Do them **in order**. Each builds on the previous.

---

## How to Open in Google Colab

1. Go to [colab.research.google.com](https://colab.research.google.com)
2. Click **File → Open Notebook → GitHub**
3. Paste the repo URL and select the notebook

Or download the `.ipynb` file and open via **File → Upload Notebook**.

---

## How to Use Each Notebook

Each section has the same pattern:

- **Demo code** — Read it and run it. The expected output is shown in comments.
- **YOUR TURN** — Fill in the `___` blanks. The structure is given — you add the logic.
- **CHALLENGE (last section)** — Harder task. Optional. Do it if you have time.

> If a section takes more than 20 minutes, move on. You can come back later.
